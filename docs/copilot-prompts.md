# GitHub Copilot Chat用プロンプト集

GitHub Copilot Chat で効果的にコードを生成するためのプロンプトテンプレート集です。

---

## ページ作成プロンプト

### ログインページを作る

```
このプロジェクトのWindows Formsコンポーネントを使って、ログインページを作成してください。

要件:
- FlowLayoutを使用して縦方向に配置
- ユーザー名入力欄（TextBox）
- パスワード入力欄（TextBox、type="password"）
- 「ログイン状態を保持」チェックボックス
- ログインボタン（primary variant）とキャンセルボタン
- ボタンは右寄せ（justify="end"）

.github/copilot-instructions.md と docs/copilot-examples.md を参照して、
このプロジェクトのスタイルに合わせてください。
```

### ユーザープロフィール編集画面を作る

```
このプロジェクトのコンポーネントを使って、ユーザープロフィール編集画面を作成してください。

要件:

- 以下のフィールドを含む:

/component-catalog を参照してコンポーネントのプロパティを確認してください。
```

## プロンプトのコツ

### ✅ 効果的なプロンプト

1. **具体的な要件を列挙**
   ```
   要件:
   - FlowLayoutを使用
   - ボタンは右寄せ
   - 間隔は10px
   ```

2. **参照ドキュメントを指定**
   ```
   /component-catalog を参照
   docs/copilot-examples.md のパターンを使用
   ```

3. **既存コードを参考に指定**
   ```
   templates/login-form.astro と同じスタイルで
   ```

### ❌ 避けるべきプロンプト

1. **曖昧な指示**
   ```
   ログイン画面を作って
   ```
   → どのコンポーネントを使うか、どんなレイアウトか不明確

2. **プロジェクトスタイルを無視**
   ```
   Tailwind CSSを使って作って
   ```
   → このプロジェクトはTailwindを使っていない

3. **過度に複雑な要求**
   ```
   完全なECサイトを一度に作って
   ```
   → 段階的に作成する方が効果的

## まとめ

- プロジェクトのドキュメント（`.github/copilot-instructions.md`、`docs/copilot-examples.md`）を活用
- 具体的な要件を列挙
- 既存のパターンやテンプレートを参考に指定
- 段階的にコードを生成・改善

これらのプロンプトを活用して、GitHub Copilotで効率的に開発を進めましょう！
