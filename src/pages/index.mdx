# Astro Components

Astroで作る再利用可能なコンポーネント集です。Windows Forms風のUIコンポーネントを作成しています。

## コンポーネント一覧

### 基本コンポーネント
- [Button（ボタン）](/button-demo) - Windows Forms風のボタンコンポーネント
- [MessageBox（メッセージボックス）](/messagebox-demo) - ダイアログ形式のメッセージ表示

### フォームコンポーネント
- [Label（ラベル）](/label-demo) - テキスト表示用ラベル
- [TextBox（テキストボックス）](/textbox-demo) - テキスト入力フィールド
- [CheckBox（チェックボックス）](/checkbox-demo) - ON/OFF選択
- [ComboBox（コンボボックス）](/combobox-demo) - ドロップダウンリスト

### レイアウトコンポーネント
- [GroupBox（グループボックス）](/groupbox-demo) - コンポーネントをグループ化

### メニューコンポーネント
- [Menu（メニュー）](/menu-demo) - MenuBar、Menu、MenuItem、ContextMenuコンポーネント

### 実用例
- [ログインフォーム](/login-form) - コンポーネントを組み合わせた実用的なフォーム
