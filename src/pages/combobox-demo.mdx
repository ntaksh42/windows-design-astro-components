# Windows Forms風コンボボックス

Windows Formsアプリケーションで使用されるようなドロップダウンリストコンポーネントです。複数の選択肢から1つを選ぶ際に使用します。

## 基本的なコンボボックス

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">
  <ComboBox options={['オプション1', 'オプション2', 'オプション3']} />
</div>

```astro
<ComboBox options={['オプション1', 'オプション2', 'オプション3']} />
```

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `options` | string[] | [] | 選択肢の配列 |
| `selectedValue` | string | '' | 選択されている値 |
| `placeholder` | string | '選択してください' | プレースホルダー |
| `width` | string | '200px' | 幅（CSS形式） |
| `disabled` | boolean | false | 無効化 |
| `name` | string | '' | name属性 |
| `id` | string | '' | id属性 |

## CSS :hover の仕組み

コンボボックスのホバー効果もCSSの `:hover` 疑似クラスで実装されています。

```css
.combobox:hover .combobox-display {
  border-color: #0078d4;
}

.combobox-item:hover {
  background: #0078d4;
  color: #fff;
}
```

**ポイント:**
- `:hover` でカーソルを当てた時のスタイルを定義
- コンボボックス本体のボーダー色を変更
- ドロップダウンアイテムのホバー時に背景色を青に変更

## 使い方のヒント

**単純なドロップダウン:**
```astro
<ComboBox options={['オプション1', 'オプション2', 'オプション3']} />
```

**初期値を設定:**
```astro
<ComboBox options={['東京', '大阪', '名古屋']} selectedValue="東京" />
```

**フォームで使用:**
```astro
<ComboBox
  name="category"
  options={['カテゴリー1', 'カテゴリー2']}
  placeholder="カテゴリーを選択"
/>
```

**試してみましょう**: コンボボックスにカーソルを当ててみてください！ドロップダウンが表示され、各アイテムのホバー効果も確認できます。

## 注意事項

このコンポーネントは視覚的なデモンストレーション用です。実際のインタラクティブな動作（クリックで選択を変更するなど）にはJavaScriptが必要です。Astroでインタラクティブな機能を実装する場合は、クライアントサイドのフレームワーク（React、Vue、Svelteなど）を統合するか、`<script>`タグでJavaScriptを追加してください。
