# RadioButton デモ

Windows Forms風のラジオボタンコンポーネントです。複数の選択肢から1つだけ選択する場合に使用します。

## 基本的な使い方

同じ`name`属性を持つラジオボタンの中から、1つだけ選択できます。

<div style="margin: 20px 0;">
  <RadioButton name="size" value="small" label="小" />
  <br />
  <RadioButton name="size" value="medium" label="中" checked />
  <br />
  <RadioButton name="size" value="large" label="大" />
</div>

```astro
<RadioButton name="size" value="small" label="小" />
<RadioButton name="size" value="medium" label="中" checked />
<RadioButton name="size" value="large" label="大" />
```

## グループ化の例

GroupBoxと組み合わせて使うことで、視覚的にグループ化できます。

## Props

| プロパティ | 型 | デフォルト | 説明 |
|----------|------|-----------|------|
| `label` | `string` | `''` | ラベルテキスト |
| `name` | `string` | **必須** | グループ名（同じnameの中から1つだけ選択可能） |
| `value` | `string` | **必須** | ラジオボタンの値 |
| `checked` | `boolean` | `false` | 選択状態 |
| `disabled` | `boolean` | `false` | 無効化状態 |

## 使用上の注意

- 同じ`name`属性を持つラジオボタンは、グループとして扱われます
- 各グループ内で1つだけ選択可能です
- 複数選択が必要な場合は、CheckBoxコンポーネントを使用してください
