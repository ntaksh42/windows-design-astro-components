# Windows Forms風パネル

Windows Formsアプリケーションで使用されるようなパネルコンポーネントです。他のコントロールをグループ化するコンテナとして機能します。

## 基本的なパネル

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">
  <Panel width={300} height={200} borderStyle="fixed-single" padding={10}>
    <p>これはパネル内のコンテンツです。</p>
    <Button label="ボタン" />
  </Panel>
</div>

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `borderStyle` | 'none' \| 'fixed-single' \| 'fixed-3d' | 'none' | ボーダーのスタイル |
| `width` | number \| string | 'auto' | パネルの幅（数値の場合はpx単位） |
| `height` | number \| string | 'auto' | パネルの高さ（数値の場合はpx単位） |
| `padding` | number | 0 | パディング（px） |
| `backgroundColor` | string | 'transparent' | 背景色 |
| `autoScroll` | boolean | false | 自動スクロールを有効にする |

## 使い方のヒント

### よく使う組み合わせ

**フォームのセクション分け:**
```astro
<Panel width={400} borderStyle="fixed-single" padding={15}>
  <h3>個人情報</h3>
  <!-- フォームフィールド -->
</Panel>
```

**サイドバーコンテナ:**
```astro
<Panel width={250} height="100%" borderStyle="fixed-3d" padding={10}>
  <!-- サイドバーコンテンツ -->
</Panel>
```

**スクロール可能なコンテンツ領域:**
```astro
<Panel width={600} height={400} borderStyle="fixed-single" autoScroll>
  <!-- 長いコンテンツ -->
</Panel>
```

## デザインパターン

- **ボーダーなし**: 視覚的なグループ化だけが必要な場合
- **単一線**: 明確な境界が必要な場合
- **3Dボーダー**: より立体的な外観が必要な場合
- **自動スクロール**: コンテンツが固定高さを超える可能性がある場合
