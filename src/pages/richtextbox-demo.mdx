# Windows Forms風リッチテキストボックス

複数行のテキスト入力に対応したリッチテキストボックスコンポーネントです。

## 基本的なリッチテキストボックス

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">
  <RichTextBox width={400} height={200} placeholder="テキストを入力してください..." />
</div>

```astro
<RichTextBox width={400} height={200} placeholder="テキストを入力してください..." />
```

## サイズバリエーション

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0; display: flex; flex-direction: column; gap: 1rem;">
  <RichTextBox width={300} height={150} placeholder="小（300x150）" />
  <RichTextBox width={500} height={250} placeholder="大（500x250）" />
</div>

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `value` | string | '' | 初期テキスト |
| `placeholder` | string | '' | プレースホルダー |
| `readOnly` | boolean | false | 読み取り専用 |
| `disabled` | boolean | false | 無効化状態 |
| `width` | number | 300 | 幅（px） |
| `height` | number | 200 | 高さ（px） |
