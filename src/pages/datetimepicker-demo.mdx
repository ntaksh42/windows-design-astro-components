## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `format` | 'long' \| 'short' \| 'time' \| 'custom' | 'short' | 表示フォーマット |
| `value` | string | '' | 初期値（YYYY-MM-DD形式） |
| `minDate` | string | - | 最小日付（YYYY-MM-DD形式） |
| `maxDate` | string | - | 最大日付（YYYY-MM-DD形式） |
| `disabled` | boolean | false | 無効化状態 |
| `width` | number | 200 | 幅（px） |
| `name` | string | - | name属性 |
| `id` | string | - | id属性 |

## フォーマットの説明

- **short**: 短い日付形式（例: 2025/11/24）
- **long**: 長い日付形式（例: 2025年11月24日）
- **time**: 時刻のみ（例: 14:30）
- **custom**: カスタムフォーマット
