# Windows Forms風チェックボックス

Windows Formsアプリケーションで使用されるようなチェックボックスコンポーネントです。ON/OFF選択や複数の選択肢から選ぶ際に使用します。

## 基本的なチェックボックス

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">
  <CheckBox label="チェックボックス" />
</div>

```astro
<CheckBox label="チェックボックス" />
```

  <CheckBox label="チェック済み" checked={true} />
</div>


## 不定状態

不定状態は、部分的に選択されている場合などに使用します。

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0; display: flex; flex-direction: column; gap: 1rem;">
  <CheckBox label="未チェック" />
  <CheckBox label="チェック済み" checked={true} />
  <CheckBox label="不定状態" indeterminate={true} />
</div>

```astro
<CheckBox label="未チェック" />
<CheckBox label="チェック済み" checked={true} />
<CheckBox label="不定状態" indeterminate={true} />
```

**説明:** 不定状態は、例えば「すべて選択」チェックボックスで一部のみ選択されている時などに使用します。

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `label` | string | '' | ラベルテキスト |
| `checked` | boolean | false | チェック状態 |
| `indeterminate` | boolean | false | 不定状態 |
| `disabled` | boolean | false | 無効化 |
| `name` | string | undefined | name属性 |
| `id` | string | undefined | id属性 |
| `value` | string | undefined | value属性 |

## CSS :hover の仕組み

チェックボックスのホバー効果もCSSの `:hover` 疑似クラスで実装されています。

```css
.checkbox-container:hover:not(.checkbox-disabled) .checkbox-box {
  border-color: #7eb4ea;
  background: #e5f3ff;
}
```

**ポイント:**
- `:hover` でカーソルを当てた時のスタイルを定義
- `:not(.checkbox-disabled)` で無効化されているチェックボックスには適用しない
- チェックボックス本体の背景色とボーダー色を変更

**フォームでの使用:**


**カスタムラベル:**
```astro

</CheckBox>
```
