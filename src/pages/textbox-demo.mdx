## 必須入力

`required`プロパティを使うと、フォーム送信時に入力チェックが行われます。

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">

    <div style="margin-bottom: 1rem;">
      <div style="margin-bottom: 0.5rem;">名前（必須）</div>
      <TextBox name="name" required placeholder="名前を入力してください" />
    </div>
    <div style="margin-bottom: 1rem;">
      <div style="margin-bottom: 0.5rem;">メールアドレス（必須）</div>
      <TextBox type="email" name="email" required placeholder="email@example.com" width="300px" />
    </div>
    <button type="submit" style="padding: 6px 16px; background: #007acc; color: white; border: none; cursor: pointer;">
      送信
    </button>
  </form>
</div>

```astro
<form>
  <TextBox name="name" required placeholder="名前を入力してください" />
  <TextBox type="email" name="email" required placeholder="email@example.com" />
  <button type="submit">送信</button>
</form>
```

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `type` | 'text' \| 'password' \| 'email' \| 'number' | 'text' | 入力タイプ |
| `placeholder` | string | '' | プレースホルダー |
| `value` | string | '' | 初期値 |
| `multiline` | boolean | false | 複数行入力 |
| `rows` | number | 3 | 行数（multilineの場合） |
| `readOnly` | boolean | false | 読み取り専用 |
| `maxLength` | number | undefined | 最大文字数 |
| `width` | string | '200px' | 幅（CSS形式） |
| `name` | string | undefined | name属性 |
| `id` | string | undefined | id属性 |
| `required` | boolean | false | 必須入力 |
| `disabled` | boolean | false | 無効化 |
| `borderStyle` | 'fixed3D' \| 'fixedSingle' \| 'none' | 'fixed3D' | ボーダースタイル |

## CSS :focus の仕組み

テキストボックスのフォーカス効果もCSSの `:focus` 疑似クラスで実装されています。

```css
.textbox:focus {
  outline: none;
  border-color: #0078d4;
  box-shadow: inset 0 0 0 1px #0078d4;
}
```

**ポイント:**
- `:focus` でフォーカスが当たった時のスタイルを定義
- `outline: none` でデフォルトのアウトラインを削除
- `border-color` と `box-shadow` でWindows風のフォーカス表現
