# NumericUpDown デモ

Windows Forms風の数値入力コンポーネントです。上下ボタンまたはキーボードの矢印キーで数値を増減できます。

## 基本的な使い方

<div style="margin: 20px 0;">
  <NumericUpDown value={50} />
</div>

```astro
<NumericUpDown value={50} />
```

## Props

| プロパティ | 型 | デフォルト | 説明 |
|----------|------|-----------|------|
| `value` | `number` | `0` | 初期値 |
| `min` | `number` | `0` | 最小値 |
| `max` | `number` | `100` | 最大値 |
| `step` | `number` | `1` | 増減ステップ |
| `width` | `string` | `'120px'` | 幅 |
| `disabled` | `boolean` | `false` | 無効化状態 |

## キーボード操作

- **↑（上矢印）**: 値を増やす
- **↓（下矢印）**: 値を減らす
- **数値入力**: 直接入力も可能（範囲外の値は自動的に補正されます）
