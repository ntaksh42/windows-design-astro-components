## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `min` | number | 0 | 最小値 |
| `max` | number | 100 | 最大値 |
| `value` | number | 50 | 初期値 |
| `step` | number | 1 | 刻み幅 |
| `orientation` | 'horizontal' \| 'vertical' | 'horizontal' | 方向 |
| `disabled` | boolean | false | 無効化状態 |
| `width` | number | 200 | 幅（px）※horizontal時 |
| `height` | number | 200 | 高さ（px）※vertical時 |
| `name` | string | - | name属性 |
| `id` | string | - | id属性 |

## 使い方のヒント

### よく使う組み合わせ

**音量調整:**
```astro
<TrackBar min={0} max={100} value={70} width={300} />
```

**温度設定:**
```astro
<TrackBar min={16} max={30} value={24} step={1} width={280} />
```

**ズーム:**
```astro
<TrackBar min={50} max={200} value={100} step={10} width={350} />
```

**垂直スライダー:**
```astro
<TrackBar orientation="vertical" height={200} />
```

## インタラクション

- **ドラッグ**: つまみをドラッグして値を変更できます
- **クリック**: レール上をクリックしてその位置に移動できます
- **キーボード**: 矢印キーで値を調整できます（フォーカス時）
- **値表示**: 現在の値がリアルタイムで表示されます

試してみましょう: 上のスライダーをドラッグして値を変更してみてください！
