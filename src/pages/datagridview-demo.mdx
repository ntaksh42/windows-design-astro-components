# Windows Forms風データグリッド

表形式のデータを表示するデータグリッドコンポーネントです。

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `columns` | string[] | [] | カラムヘッダー |
| `rows` | string[][] | [] | データ行 |
| `width` | number | 500 | 幅（px） |
| `height` | number | 300 | 高さ（px） |

## インタラクション

- **スクロール**: データが多い場合は自動的にスクロールバーが表示されます
- **ホバー**: 行の上にマウスを置くとハイライト表示されます
