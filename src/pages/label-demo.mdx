# Windows Forms風ラベル

Windows Formsアプリケーションで使用されるようなラベルコンポーネントです。テキストの表示やフォーム要素のラベルとして使用します。

## 基本的なラベル

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">
  <Label text="これは基本的なラベルです" />
</div>

```astro
<Label text="これは基本的なラベルです" />
```

## ボーダースタイル

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0; display: flex; flex-direction: column; gap: 1rem;">
  <Label text="ボーダーなし（デフォルト）" borderStyle="none" />
  <Label text="単純なボーダー" borderStyle="fixedSingle" />
  <Label text="3D風ボーダー" borderStyle="fixed3D" />
</div>

```astro
<Label text="ボーダーなし（デフォルト）" borderStyle="none" />
<Label text="単純なボーダー" borderStyle="fixedSingle" />
<Label text="3D風ボーダー" borderStyle="fixed3D" />
```

**説明:**
- `none`: ボーダーなし（デフォルト）
- `fixedSingle`: 1pxの単純なボーダー
- `fixed3D`: 影付きの3D風ボーダー

## 自動サイズ調整

`autoSize`プロパティを使うと、テキストの長さに合わせて自動的にサイズが調整されます。

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0; display: flex; flex-direction: column; gap: 1rem;">
  <Label text="短いテキスト" autoSize borderStyle="fixedSingle" />
  <Label text="これは少し長めのテキストです" autoSize borderStyle="fixedSingle" />
  <Label text="これはとても長いテキストで、自動的にサイズが調整されます" autoSize borderStyle="fixedSingle" />
</div>

```astro
<Label text="短いテキスト" autoSize borderStyle="fixedSingle" />
<Label text="これは少し長めのテキストです" autoSize borderStyle="fixedSingle" />
```

## スロットを使ったカスタマイズ

スロットを使うと、HTMLやスタイルを含めたカスタムコンテンツを表示できます。

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0; display: flex; flex-direction: column; gap: 1rem;">
  <Label borderStyle="fixedSingle">
    <strong>太字のテキスト</strong>
  </Label>
  <Label borderStyle="fixedSingle">
    <span style="color: #e74c3c;">赤いテキスト</span>
  </Label>
  <Label borderStyle="fixedSingle">
    ✓ チェックマーク付き
  </Label>
</div>

```astro
<Label borderStyle="fixedSingle">
  <strong>太字のテキスト</strong>
</Label>
<Label borderStyle="fixedSingle">
  <span style="color: #e74c3c;">赤いテキスト</span>
</Label>
```

## フォーム要素のラベルとして使用

`htmlFor`プロパティを使うと、入力要素と関連付けることができます。ラベルをクリックすると、関連する入力要素にフォーカスが当たります。

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0;">
  <div style="display: flex; flex-direction: column; gap: 0.5rem;">
    <Label text="ユーザー名:" htmlFor="username" />
    <input type="text" id="username" style="width: 200px; padding: 4px; border: 1px solid #acacac;" />
  </div>
  <div style="display: flex; flex-direction: column; gap: 0.5rem; margin-top: 1rem;">
    <Label text="パスワード:" htmlFor="password" />
    <input type="password" id="password" style="width: 200px; padding: 4px; border: 1px solid #acacac;" />
  </div>
</div>

```astro
<Label text="ユーザー名:" htmlFor="username" />
<input type="text" id="username" />

<Label text="パスワード:" htmlFor="password" />
<input type="password" id="password" />
```

**ポイント:** ラベルをクリックすると、関連する入力要素にフォーカスが当たります。これにより、ユーザビリティが向上します。

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `text` | string | '' | 表示するテキスト |
| `textAlign` | 'left' \| 'center' \| 'right' | 'left' | テキストの配置 |
| `borderStyle` | 'none' \| 'fixedSingle' \| 'fixed3D' | 'none' | ボーダースタイル |
| `autoSize` | boolean | false | 自動サイズ調整 |
| `width` | string | undefined | 幅（CSS形式: '200px'など） |
| `htmlFor` | string | undefined | 関連する入力要素のID |

## 使い方のヒント

**単純なテキスト表示:**
```astro
<Label text="こんにちは" />
```

**フォームのラベル:**
```astro
<Label text="名前:" htmlFor="name-input" />
<input type="text" id="name-input" />
```

**情報パネル:**
```astro
<Label text="ファイル名: document.txt" borderStyle="fixed3D" width="300px" />
```

**ステータス表示:**
```astro
<Label borderStyle="fixedSingle">
  <span style="color: green;">✓ 接続成功</span>
</Label>
```
