## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `items` | string[] | [] | リストに表示する項目の配列 |
| `selectionMode` | 'single' \| 'multiple' | 'single' | 選択モード（現在は単一選択のみ実装） |
| `height` | number | 200 | リストボックスの高さ（px） |
| `width` | number | 200 | リストボックスの幅（px） |
| `disabled` | boolean | false | 無効化状態 |
| `name` | string | - | name属性 |

## 使い方のヒント

### よく使う組み合わせ

**ファイル選択ダイアログ:**
```astro
<ListBox
  items={fileList}
  height={300}
  width={400}
/>
```

**設定項目の選択:**
```astro
<ListBox
  items={['オプション1', 'オプション2', 'オプション3']}
  height={120}
  width={200}
/>
```

**コンパクトなリスト:**
```astro
<ListBox
  items={['A', 'B', 'C']}
  height={80}
  width={100}
/>
```

## インタラクション

- **クリック**: 項目をクリックすると選択状態になります
- **ホバー**: 項目の上にマウスを置くとハイライト表示されます
- **スクロール**: 項目が多い場合は自動的にスクロールバーが表示されます
- **選択**: 単一選択モードでは、1つの項目のみ選択できます
