## バリエーション

### デフォルトボタン

<div style="margin: 2rem 0; padding: 2rem; background: #f0f0f0; display: flex; gap: 1rem; flex-wrap: wrap;">
  <Button label="デフォルト" variant="default" />
  <Button label="プライマリ" variant="primary" />
  <Button label="フラット" variant="flat" />
</div>

```astro
<Button label="デフォルト" variant="default" />
<Button label="プライマリ" variant="primary" />
<Button label="フラット" variant="flat" />
```

**説明:**
- `default`: 標準ボタン（グレーのグラデーション）
- `primary`: 重要なアクション用（青色、太字）
- `flat`: フラットスタイル（背景透明、ホバー時のみ背景表示）

## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `label` | string | 'Button' | ボタンのテキスト |
| `variant` | 'default' \| 'primary' \| 'flat' | 'default' | ボタンの種類 |
| `size` | 'small' \| 'medium' \| 'large' | 'medium' | ボタンのサイズ |
| `disabled` | boolean | false | 無効化状態 |
| `type` | 'button' \| 'submit' \| 'reset' | 'button' | HTMLボタンのtype属性 |
| `fullWidth` | boolean | false | 横幅を100%にする |

## CSS :hover の仕組み

ボタンのホバー効果もCSSの `:hover` 疑似クラスで実装されています。

### ホバー時のスタイル変更

```css
.btn-default:hover:not(:disabled) {
  background: linear-gradient(to bottom, #e5f3ff 0%, #d0e9ff 100%);
  border-color: #3c7fb1;
}
```

**ポイント:**
- `:hover` でカーソルを当てた時のスタイルを定義
- `:not(:disabled)` で無効化されているボタンには適用しない
- `linear-gradient` でグラデーション背景を実現

### アクティブ状態（クリック時）

```css
.btn-default:active:not(:disabled) {
  background: linear-gradient(to bottom, #cce4f7 0%, #b8d6ed 100%);
  border-color: #2c5f8d;
}
```

`:active` 疑似クラスでボタンを押している間のスタイルを定義します。

## 使い方のヒント

### よく使う組み合わせ

**OKボタン（アクション実行）:**
```astro
<Button label="OK" variant="primary" />
```

**キャンセルボタン:**
```astro
<Button label="キャンセル" variant="default" />
```

**削除などの危険な操作（色は今後追加予定）:**
```astro
<Button label="削除" variant="default" />
```

**ツールバーのボタン:**
```astro
<Button label="保存" variant="flat" size="small" />
```

**試してみましょう**: 上のボタンにカーソルを当てたり、クリックしてみてください！ホバー効果とアクティブ効果が確認できます。
