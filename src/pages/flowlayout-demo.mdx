# FlowLayout（フローレイアウト）

子要素を自動的に横または縦に並べるコンポーネントです。AIが座標を計算せずに要素を配置できます。

## 基本的な使い方

### 横方向（水平）配置

```astro
<FlowLayout direction="horizontal" gap={10}>
  <Button>ボタン1</Button>
  <Button>ボタン2</Button>
  <Button>ボタン3</Button>
</FlowLayout>
```

<FlowLayout direction="horizontal" gap={10} padding={20} background="#f5f5f5" border={true}>
  <Button>ボタン1</Button>
  <Button>ボタン2</Button>
  <Button>ボタン3</Button>
</FlowLayout>

---

## プロパティ

| プロパティ | 型 | デフォルト | 説明 |
|----------|-----|----------|------|
| `direction` | `'horizontal' \| 'vertical'` | `'horizontal'` | 配置方向 |
| `gap` | `number` | `8` | 要素間の間隔（ピクセル） |
| `align` | `'start' \| 'center' \| 'end' \| 'stretch'` | `'start'` | 交差軸方向の配置 |
| `justify` | `'start' \| 'center' \| 'end' \| 'space-between' \| 'space-around' \| 'space-evenly'` | `'start'` | 主軸方向の配置 |
| `wrap` | `boolean` | `false` | 折り返しを許可するか |
| `padding` | `number` | `0` | 内側の余白（ピクセル） |
| `background` | `string` | `'transparent'` | 背景色 |
| `border` | `boolean` | `false` | ボーダーを表示するか |
| `width` | `number \| string` | `'auto'` | 幅 |
| `height` | `number \| string` | `'auto'` | 高さ |

## AI向けの使い方ヒント

FlowLayoutを使えば、AIは座標を計算せずに要素を並べられます：

### 横に並べる場合
```json
{
  "type": "FlowLayout",
  "props": {"direction": "horizontal", "gap": 10},
  "children": [
    {"type": "Button", "text": "保存"},
    {"type": "Button", "text": "キャンセル"}
  ]
}
```

### 縦に並べる場合
```json
{
  "type": "FlowLayout",
  "props": {"direction": "vertical", "gap": 10},
  "children": [
    {"type": "Label", "text": "名前:"},
    {"type": "TextBox", "placeholder": "入力してください"}
  ]
}
```

### フォーム全体の例
```json
{
  "type": "FlowLayout",
  "props": {"direction": "vertical", "gap": 15, "padding": 20},
  "children": [
    {
      "type": "FlowLayout",
      "props": {"direction": "vertical", "gap": 5},
      "children": [
        {"type": "Label", "text": "ユーザー名:"},
        {"type": "TextBox"}
      ]
    },
    {
      "type": "FlowLayout",
      "props": {"direction": "vertical", "gap": 5},
      "children": [
        {"type": "Label", "text": "パスワード:"},
        {"type": "TextBox", "type": "password"}
      ]
    },
    {
      "type": "FlowLayout",
      "props": {"gap": 10, "justify": "end"},
      "children": [
        {"type": "Button", "text": "キャンセル"},
        {"type": "Button", "text": "ログイン", "variant": "primary"}
      ]
    }
  ]
}
```

**メリット**:
- 座標計算が不要
- レスポンシブに対応しやすい
- コードが読みやすい
- メンテナンスが簡単

Containerと組み合わせれば、さらに柔軟なレイアウトが可能です！
