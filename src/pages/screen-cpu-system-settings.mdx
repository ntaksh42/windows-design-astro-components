## 説明

このページは「CPUシステム設定」ダイアログのUIを再現したものです。

### 構成要素

- **左側パネル**: TreeViewで設定カテゴリを階層表示
  - プログラムの設定
  - システムの設定
- **右側パネル**: 「その他の設定」の設定項目
  - プロジェクト言語設定（ComboBox）
  - ラダープログラム編集方式（CheckBox）
  - ラダー列数設定（無効化されたボタン）
  - OK/キャンセル/適用ボタン

### 使用コンポーネント

- SplitContainer（垂直分割レイアウト）
- TreeView（階層メニュー）
- ComboBox（ドロップダウンリスト）
- CheckBox（チェックボックス）
- Button（ボタン）
- Label（ラベル）
- FlowLayout（フレックスレイアウト）
