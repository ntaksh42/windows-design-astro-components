## プロパティ一覧

| プロパティ | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `src` | string | '' | 画像のURL |
| `alt` | string | '' | 代替テキスト |
| `sizeMode` | 'normal' \| 'stretch' \| 'auto-size' \| 'center' \| 'zoom' | 'normal' | 画像のサイズモード |
| `width` | number | 100 | 幅（px） |
| `height` | number | 100 | 高さ（px） |
| `borderStyle` | 'none' \| 'fixed-single' \| 'fixed-3d' | 'fixed-single' | ボーダースタイル |

## サイズモードの説明

- **normal**: 画像をそのままのサイズで表示（はみ出す部分は切り取られる）
- **stretch**: 画像をボックスに合わせて引き伸ばす（アスペクト比は保持されない）
- **auto-size**: ボックスを画像サイズに合わせる
- **center**: 画像を中央に配置（はみ出す部分は切り取られる）
- **zoom**: アスペクト比を保ったまま、ボックスに収まるように拡大/縮小

**プロフィール写真:**
```astro

  src="/profile.jpg"
  width={100}
  height={100}
  sizeMode="zoom"
  borderStyle="fixed-3d"
/>
```

**アイコン表示:**
```astro
<PictureBox
  src="/icon.png"
  width={32}
  height={32}
  sizeMode="center"
  borderStyle="none"
/>
```

**サムネイル:**
```astro
<PictureBox
  src="/thumbnail.jpg"
  width={120}
  height={90}
  sizeMode="zoom"
  borderStyle="fixed-single"
/>
```
