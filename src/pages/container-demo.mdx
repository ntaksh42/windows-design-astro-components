# Container（コンテナ）

座標を指定して要素を自由に配置できるコンポーネントです。AIがUIを「描く」ために便利です。

## 基本的な使い方

### 絶対座標配置

```astro
<Container x={100} y={50} width={200} height={100}>
  <p>座標(100, 50)に配置</p>
</Container>
```

## 相対配置モード

`relative={true}` を指定すると、通常のフロー配置になります。

```astro
<Container relative={true} padding={20} background="#f5f5f5" border={true}>
  <p>相対配置</p>
</Container>
```

<Container relative={true} padding={20} background="#f5f5f5" border={true}>
  <Label>これは相対配置のコンテナです</Label>
</Container>

## プロパティ

| プロパティ | 型 | デフォルト | 説明 |
|----------|-----|----------|------|
| `x` | `number` | `0` | X座標（ピクセル） |
| `y` | `number` | `0` | Y座標（ピクセル） |
| `width` | `number \| string` | `'auto'` | 幅（数値の場合はpx、文字列なら'100%'等も可） |
| `height` | `number \| string` | `'auto'` | 高さ（数値の場合はpx、文字列なら'100%'等も可） |
| `padding` | `number` | `0` | 内側の余白（ピクセル） |
| `background` | `string` | `'transparent'` | 背景色 |
| `border` | `boolean` | `false` | ボーダーを表示するか |
| `relative` | `boolean` | `false` | 相対配置モード（trueの場合、x/yは無視される） |

## AI向けの使い方ヒント

AIがこのコンポーネントでUIを描く場合：

1. **親要素に `position: relative` を設定** - 座標の基準点を作る
2. **座標を計算** - 各要素の位置を決定
3. **Containerで配置** - x, y, width, height を指定

### JSONから生成する例

```json
{
  "containers": [
    {"x": 20, "y": 20, "width": 200, "component": "Label", "text": "名前:"},
    {"x": 20, "y": 45, "width": 300, "component": "TextBox"},
    {"x": 20, "y": 90, "width": 100, "component": "Button", "text": "送信"}
  ]
}
```

このようなJSONをContainerと各コンポーネントに変換すれば、AIが簡単にUIを生成できます！
